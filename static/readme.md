# Static Assets Usage

## Installation

1. Install the Astro UXDS Static Assets

    ` npm i --save-dev astro-components-css-assets`

2. Import
   
    `import {astrocore} from 'astrocore';`

3. Relative Path
   
   `<link href="./node_modules/astro-static-css-assets/css/[Filename]>`
