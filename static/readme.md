# Static Assets Usage

## Installation

1. Install the Astro UXDS Static Assets

    `npm i @astrouxds/rux-static `

2. Import
   
    `import '@astrouxds/rux-static';`

3. Relative Path
   
   `<link href="./node_modules/astro-static-css-assets/css/[Filename]>`
